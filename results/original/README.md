# Original Results

This folder contains tables, logs, figures, and run outputs copied from the
original development workspace.

## How To Read This Folder

Start with the curated summaries first:

- `results/README.md`
- `results/tables/seg_crop_segR_lab.json`
- `results/tables/loc_detector.json`
- `experiments/best_current/README.md`
- `experiments/rejected/README.md`

Then use files here only to audit a specific historical claim, log, montage, or
ablation result.

## Important Note

This folder is intentionally broad. It includes successful baselines, negative
experiments, smoke runs, logs, and qualitative figures. It is not the minimal
set of files needed to run the final deliverable.
