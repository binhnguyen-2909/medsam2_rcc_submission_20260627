No notebook is required for the submitted reproducible workflow.

Use the command-line entrypoints instead:

- current inference: `src/inference/predict_seg_crop.py`

If notebooks are added later, keep them as exploratory analysis only and mirror
any reproducible command back into `experiments/runs/`.
